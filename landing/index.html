Code Sample

Looks familiar.
Works immediately.

Idiomatic API design. If you've used Stripe or Resend, you already know how this works.

curl -X POST https://api.sendnex.xyz/v1/emails \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "from": "hello@yourdomain.com",
    "to": "user@example.com",
    "subject": "Welcome to the app",
    "html": "<h1>You'\''re in.</h1><p>Thanks for signing up.</p>",
    "tags": ["welcome", "onboarding"]
  }'

# Response
{
  "messageId": "msg_01HXYZ9QK2NPRT4VWCBD83EF7G",
  "status": "queued",
  "timestamp": "2025-10-14T09:41:22.314Z"
}
import { SendNextClient } from '@sendnext/sdk';

const client = new SendNextClient({
  apiKey: process.env.SENDNEXT_API_KEY
});

const result = await client.email.send({
  from: 'hello@yourdomain.com',
  to: 'user@example.com',
  subject: 'Welcome to the app',
  html: '<h1>You\'re in.</h1><p>Thanks for signing up.</p>',
  tags: ['welcome', 'onboarding'],
});

console.log(result.messageId);
// msg_01HXYZ9QK2NPRT4VWCBD83EF7G
