Overview ↔ Email Logs
Click stat card (bounce rate, delivery rate) → Filtered log view
Click recent email row → Email detail view
Click alert → Relevant management page
Overview ↔ Analytics
Click any stat card → Analytics filtered to that metric
30-day chart on overview is a summary of full analytics
Email Logs ↔ Email Detail
Click any email row → Full email detail view
Detail view shows status timeline, bounce reason, template used
Back button returns to filtered log view
Email Detail ↔ Suppressions
Bounced email detail links to suppression entry
"View in Suppression List" link on bounced/complained emails
Email Detail ↔ Templates
Email detail shows which template was used
"View Template" link navigates to template editor
Templates ↔ Template Editor
Click template row → Template editor/detail
"Create Template" → Blank editor
Save in editor → Return to template list
Templates ↔ API Docs
Template editor shows template_id for API use
API docs reference template sending examples
Analytics ↔ Email Logs
Drill into template performance → Filtered email logs
Click chart data point → Logs for that day
Analytics ↔ Templates
Template performance chart links to template editor
"Edit Template" from analytics drilldown
Domains ↔ Domain Detail
Click domain row → DNS records and verification detail
"Verify Now" triggers verification from detail view
Domains ↔ Overview
Domain verification failure alerts on overview
Click alert → Domain detail page
API Keys ↔ Analytics
Per-key sending stats shown in analytics
Click key name in analytics → API key detail
